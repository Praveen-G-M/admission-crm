\# 🎓 Admission Management CRM



A full-stack Admission Management System application streamlines the college admission workflow including authentication, institution management, applicant tracking, seat allocation, and document verification.



---



\## 🚀 Tech Stack



\### 🔹 Backend

\- ASP.NET Core Web API (.NET)

\- Entity Framework Core

\- SQL Server

\- JWT Authentication

\- BCrypt Password Hashing

\- Swagger (OpenAPI)



\### 🔹 Frontend

\- React (Vite)

\- Tailwind CSS

\- Axios

\- React Router



---



\## ✨ Key Features



\- 🔐 Secure JWT authentication  

\- 👥 Role-based access control  

\- 🏫 Institution management  

\- 📊 Dashboard overview  

\- 🧾 Applicant management  

\- 🪑 Seat matrix management  

\- 📁 Document tracking  

\- 🔄 End-to-end admission workflow
